# Contributors

This file lists everyone who has contributed to Attestplane. It is a thank-you,
not a legal copyright assignment: this project uses the **Developer Certificate
of Origin (DCO)**, not a Contributor License Agreement (CLA), so contributors
retain full copyright in their work and license it under Apache-2.0 by the
`Signed-off-by:` line on every commit. See [`DCO.txt`](DCO.txt) and
[`CONTRIBUTING.md`](CONTRIBUTING.md).

The authoritative list of contributors is also the git history itself
(`git log --format='%an <%ae>' | sort -u`). This file exists for human
recognition and onboarding context.

## Adding yourself

If your contribution has been merged and you would like to be listed here,
open a PR adding your row. We trust the git history as the ground truth for
who has contributed; this file is human curation on top of that. We will not
unilaterally add your name without a request, in case you prefer to remain
in the git history only.

If your handle changes or you want to be de-listed, open a PR — we will
honor any reasonable request.
