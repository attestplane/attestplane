# Anchoring Quarantine Fixtures

These frozen bundles exercise the claim-safe quarantine path used by
`attestplane verify --json`. The JSON files are intentionally minimal and
mirror the final bundle surface only; the live TSA failure mode that
produced them is recorded by their filenames.
