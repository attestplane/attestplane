# v1.5.5

`v1.5.5` is an automated suffix-free stable package cut from autodev-train.

## User-Visible Delta

The user-visible delta in `v1.5.5` is the stable package cut itself: the
release publishes suffix-free Python and npm package versions through the
stable release workflow, advances PyPI default installs, and advances npm
`latest`. It does not move npm `ca`, and it stays within the documented
release surface without claiming compliance, production readiness, or
registry policy changes beyond the stable path.
