# v1.5.9

`v1.5.9` is an automated suffix-free stable package cut from autodev-train.

## User-Visible Delta

The single user-visible change in `v1.5.9` is release-planning integrity:
suffix-free stable package cuts now require Opus consultation before the stable
planning flow proceeds. This documents the planning checkpoint for the
autodev-train stable release path; it does not imply any SDK, verifier, schema,
artifact, registry, signing, compliance, production-readiness, or provenance
behavior changed.

Planning context: [Issue #91](https://github.com/attestplane/attestplane/issues/91);
planned docs task:
[Issue #94](https://github.com/attestplane/attestplane/issues/94).

## Highlights

- Publishes suffix-free Python and npm package versions through GitHub `release-cd`.
- Advances PyPI default installs because PyPI has no dist-tag concept.
- Advances npm `latest` through the stable release workflow.
- Does not move npm `ca`; CA remains a separate manual dist-tag decision.
