# v1.6.1

`v1.6.1` is an automated suffix-free stable package cut from autodev-train.

## User-Visible Delta

- The audited `v1.6.1` boundary has one product-impacting change:
  [`f181c6d`](https://github.com/attestplane/attestplanecommit/f181c6d4af6d6b792e53b17c8d5426cb2c9d805f)
  fixes planned-issue refetch after creation so downstream runners see the
  newly created planned-task issues on the next fetch.
- Tag `v1.6.1` itself still points at
  [`f2a55d4`](https://github.com/attestplane/attestplanecommit/f2a55d4baea9d27bfac2ea40fd835c0f3e237048)
  (`chore(release): prepare v1.6.1`), so this note records the boundary
  rather than moving the tag.

## Infrastructure

- `ci: proxy architecture audit runner network` is CI-only release-train
  hygiene. It does not change SDK, CLI, verifier, schema, package, registry,
  signing, or claim-safety behavior.

## Highlights

- Publishes suffix-free Python and npm package versions through GitHub `release-cd`.
- Advances PyPI default installs because PyPI has no dist-tag concept.
- Advances npm `latest` through the stable release workflow.
- Does not move npm `ca`; CA remains a separate manual dist-tag decision.
