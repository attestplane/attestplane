# v1.7.0

`v1.7.0` is an automated suffix-free stable package cut from autodev-train.

## Changes Since Previous Stable

- feat: require non-empty proof bundles on demand
- feat: add `attestplane.sdk.ProofBundleBuilder.minimal(subject_digest, signer)` and typed
  `EmptyProofBundleError` / `IncompleteProofBundleError` migration path for strict
  proof-bundle rejection codes.
- fix: block productless stable releases
- fix: prioritize product autodev plans

## Highlights

- Publishes suffix-free Python and npm package versions through GitHub `release-cd`.
- Advances PyPI default installs because PyPI has no dist-tag concept.
- Advances npm `latest` through the stable release workflow.
- Does not move npm `ca`; CA remains a separate manual dist-tag decision.
