# v1.5.7

`v1.5.7` is an automated suffix-free stable package cut from autodev-train.

Planning context: this note is linked to source planning issue
[#82](https://github.com/attestplane/attestplane/issues/82) (plan ID:
`95b8871fa762c209`).

## User-Visible Delta

The user-visible delta in `v1.5.7` consists of fixes to the release-planning
workflow: planned issues are now reliably reloaded from GitHub, and open issues
are included in the planning scope so that in-flight work is not skipped. A
concurrent CI change ignores transient scorecard link failures to reduce false
positive signal noise. The release remains within the documented stable train
surface without claiming compliance, production readiness, or registry policy
changes beyond the stable path.

## Changes Since Previous Stable

- ci: ignore transient scorecard link failures
- chore(release): prepare v1.5.7
- fix: reload planned issues from github
- fix: include open issues in release planning

## Highlights

- Publishes suffix-free Python and npm package versions through GitHub `release-cd`.
- Advances PyPI default installs because PyPI has no dist-tag concept.
- Advances npm `latest` through the stable release workflow.
- Does not move npm `ca`; CA remains a separate manual dist-tag decision.
