# v0.8.0-beta.0

`v0.8.0-beta.0` opens the Attestplane public beta prerelease line.

## Highlights

- Promotes the dual-SDK substrate from alpha to beta prerelease posture.
- Keeps npm `latest` unchanged and publishes the TypeScript package through the `beta` dist-tag.
- Preserves deterministic verifier, release-artifact, registry-verification, and claim-safety gates.
- Re-runs the full PyPI and npm publication path for the beta boundary.

## Release Boundary

This beta is a prerelease. It is not GA, not production-ready, not a compliance certification, and not legal advice.

API surfaces are entering a freeze window, but breaking changes may still occur before GA when required by verifier correctness, evidence-format safety, or claim-boundary enforcement.

## Explicit Boundaries

This release does not claim:

- EU AI Act compliance,
- GDPR compliance,
- legal certification,
- production readiness,
- certified provenance,
- SLSA L3,
- production-grade supply-chain security, or
- long-term archival trust guarantees.
