## Publish Method

GitHub Actions `workflow_dispatch` + OIDC trusted publishing.

- `publish-python.yml` — `pypa/gh-action-pypi-publish@v1.14.0` with
  `permissions: id-token: write` and `environment: pypi` on the
  publish job. No PyPI API token used.
- `publish-typescript.yml` — `npm publish --provenance --access public
  --tag alpha` with `permissions: id-token: write` for npm provenance
  attestation; npm auth via `secrets.NPM_TOKEN`.
