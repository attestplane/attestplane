# Non-Goals

Attestplane is an open-source evidence substrate and verifier surface. The
alpha line intentionally does not provide or claim:

- legal compliance certification,
- EU AI Act compliance,
- GDPR compliance,
- DORA compliance,
- production readiness,
- autonomous runtime governance,
- hosted API authority,
- closed verifier authority,
- complete sidecar deletion across external processors,
- SLSA L3,
- production-grade supply-chain security, or
- long-term archival trust guarantees.

The project can record evidence that helps a deployer build an auditable
system. It does not replace deployer-specific legal review, retention policy,
security controls, or operational governance.
